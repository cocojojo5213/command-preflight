Command Preflight for Windows

1. Extract every file from the ZIP archive.
2. Double-click INSTALL.cmd.
3. Choose whether to enable the optional read-only community lookup.
4. Open a new terminal after setup finishes.

Do not double-click command-preflight.exe to install it. The EXE is the command-line
program; opening it directly only shows usage and waits for Enter so the window does
not disappear immediately.
